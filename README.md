# Парсер смартфонов

**Задача**

Разработать парсер, который будет собирать информацию о версиях операционных систем в топ-100 смартфонах с самым высоким рейтингом пользователей в каталоге ozon.ru.

**Пример вывода**
```
os       version    counts
Android  11         41
         10         19
iOS      14         13
         15          9
         13          7
Android  12          3
```



**Используемые технологии:**
- Scrapy 2.7.1
- Selenium 4.6.0
- Pandas 1.5.1
